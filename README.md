# Supply Chain Disruption Prediction

An AI-powered web application that predicts potential supply chain disruptions using Machine Learning. The system analyzes logistics and shipment data, estimates disruption risk, visualizes important trends, and provides AI-driven recommendations to help businesses make proactive supply chain decisions.

## Features
- Machine Learning-based disruption prediction
- Risk level classification (High/Low)
- Interactive data visualizations
- AI-generated recommendations
- Responsive Flask web application
- Easy-to-use interface

## Tech Stack
- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- HTML, CSS, JavaScript
